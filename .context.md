# 🧠 AI SYSTEM INSTRUCTIONS FOR TTS-GATEWAY-SERVICE
1. **Teknoloji:** Rust 1.84+ (Tokio, Tonic).
2. **Multi-Engine Routing:** Gelen isteğin `voice_id` veya `provider` parametresine göre hedef motoru (Coqui, MMS vb.) dinamik seçmelidir. Spagetti koddan (hardcoded if-else) kaçınılmalı, modüler tasarım uygulanmalıdır.
3. **mTLS Strict:** Insecure gRPC fallback YASAKTIR.
4. **Loglama:** Tüm loglar SUTS v4.0 JSON formatında olmalı ve `tracing` crate kullanılmalıdır. `unwrap()` kullanımı minimumda tutulmalı, hatalar üst katmana döndürülmelidir.
